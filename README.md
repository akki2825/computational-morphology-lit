## Topics Covered

| Topic | Description |
|-------|-------------|
| Morphological Inflection | Generating inflected word forms from lemmas and feature bundles |
| Morphological Segmentation | Splitting words into morphemes |
| Paradigm Completion | Filling in missing cells of inflectional paradigms |
| Morphological Analysis & Tagging | Assigning morphological feature bundles to word forms |
| Lemmatization | Mapping inflected forms back to their lemma |
| Morphological Typology | Computational approaches to morphological diversity |
| Low-Resource Morphology | Cross-lingual transfer and data augmentation for morphology |
| Cognitive Modeling of Morphology | Neural networks as models of human morphological processing |
| Subword Tokenization | BPE, SentencePiece, and morphologically-informed tokenization |
| Character-Level Models | Character-aware neural models for morphological processing |
| Finite-State Morphology | Transducer-based approaches to morphological analysis |
| Computational Phonology | Neural and symbolic approaches to phonological processes |

## Software and Tools

| Tool | Description |
|------|-------------|
| [Morfessor](https://github.com/aalto-speech/morfessor) | Unsupervised morphological segmentation |
| [OpenFST](https://www.openfst.org/) | Finite-state transducer library |
| XFST | Xerox finite-state tool for morphological analysis (commercial) |
| [Foma](https://fomafst.github.io/) | Finite-state compiler and library for morphological analysis |
| [HFST](https://hfst.github.io/) | Helsinki Finite-State Technology toolkit |
| [SentencePiece](https://github.com/google/sentencepiece) | Unsupervised text tokenizer and detokenizer |
| [UniMorph](https://unimorph.github.io/) | Universal morphological feature schema and inflectional paradigms |
| [Sigmorphon Data](https://github.com/sigmorphon) | Shared task data and baselines |
